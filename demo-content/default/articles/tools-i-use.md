Inspired by [uses.tech](https://uses.tech), here's what I'm working with right now.

## Editor

**VS Code** with the Catppuccin theme. I've tried switching to Zed twice and come back both times. The extension ecosystem is just too good.

## Terminal

**Ghostty** on macOS, **kitty** on Linux. I use zsh with a minimal prompt — just the directory and git branch.

## Hosting

Everything runs on **Cloudflare Pages** with D1 for the database and R2 for media. The free tier is absurdly generous.

## Daily drivers

- **Raycast** — launcher, clipboard history, snippets
- **1Password** — shared vaults with my team
- **Arc** — browser, though I hear it's on life support
- **Linear** — issue tracking for side projects
- **Claude Code** — pair programming, refactoring, the boring stuff

## This site

Built with **IndiePub** on Astro. Posts via Micropub, replies via Webmention, feeds via RSS. The whole thing is about 30 files.
